# ReactorOps

ReactorOps is the DiamondOps asset lifecycle layer: inspection, retrofit planning, refurbishment, and maintenance contracts.

## Scope (v0.1)
- Service SKU definitions
- Failure mode libraries (high-level)
- Used-asset evaluation checklists
- Integration event specs aligned to DiamondOps-Core

## Dependency
Aligned to DiamondOps-Core schemas and standards.
